> vue组件开发

## 安装

```sh
npm install @istrong/fnd-vue-component
```

## 使用

```js
import Vue from 'vue'
import VuePlugin from 'fnd-vue-component'

Vue.use(VuePlugin)
// 或者，你也可以传入可选参数
Vue.use(VuePlugin,{})
```
